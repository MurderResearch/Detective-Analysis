あらすじ

1878年、インド駐留のアーサー・モースタン大尉がボンベイからロンドンへ帰還し、その夜、旧友のショルトー少佐を訪ねた。二人はインドから持ち帰った莫大な宝石——「アグラの財宝」——の分配をめぐって激しく口論した。モースタンは激怒のさなかに心臓発作で急死した。ショルトーは殺人の疑いをかけられることを恐れ、遺体を隠し、財宝を独り占めにした。モースタンの娘メアリーはその後、父に二度と会うことがなかった。

六年後、ショルトー少佐は臨終の床で双子の息子サディアスとバーソロミューにすべてを告白した。財宝の隠し場所を明かそうとしたまさにその瞬間、窓の外に凶相のひげ面が現れた——アンダマン諸島の刑務所から脱獄し、長年にわたって財宝を追い求めていたジョナサン・スモールであった——ショルトーはその衝撃で息絶え、秘密の最後の一言を持ったまま逝った。

1888年、メアリー・モースタンは差出人不明の謎めいた手紙を受け取り、ライシアム劇場の前に招かれた。ホームズとワトスンが同行し、そこで不安を隠せないサディアス・ショルトーと知り合った。サディアスは、弟のバーソロミューが屋根裏でその財宝を発見したばかりだと告げた。

しかし一行がポンディシェリー・ロッジに駆けつけると、バーソロミュー・ショルトーはすでに息絶えていた——顔には恐ろしい苦悶の笑みが張り付き、頭皮には小さな毒針が刺さっており、財宝は跡形もなく消えていた。卓上には一枚のメモが残されていた：「四つの署名」。

ホームズは現場をすばやく読み解いた：木製義足の痕跡、半指ほどの大きさの裸足の足跡、異国産の毒吹き矢、そして屋根から侵入したロープの痕跡。彼は二人の犯人を特定した：片足の義足を持つジョナサン・スモールと、アンダマン諸島出身の島民の共犯者。「ベイカー街不規則隊」（Baker Street Irregulars）の街の情報屋たち、鋭敏な嗅覚を持つ雑種犬トビー、そして昼夜を問わない追跡が、テムズ川上の蒸気船オーロラ号へと彼らを導いた。

夜のテムズ川での追跡は悲劇的な結末を迎えた：島民トンガが吹き矢を構えた瞬間、ホームズとワトスンが同時に引き金を引き、トンガはテムズ川に落ちて二度と浮かび上がらなかった。ジョナサン・スモールは捕縛されたが、追い詰められる寸前にすでに財宝を川底に静かに沈めていた。重い鉄の箱を開けると、中は空だった。

ワトスンは財宝が消えた瞬間、最も正直な一言を口にした：「神に感謝する！」——彼とメアリー・モースタンの間に立ちはだかっていた財富という壁が、ついに消え去ったのだ。

真相解剖

真相その一：バーソロミュー・ショルトーはどのようにして死んだのか？（howdunit）

  • 手がかり収束点：約50%
  • 答え：トンガがアンダマン諸島の原住民の毒吹き矢（blow-pipe dart）を用いて、天井の開口部からバーソロミュー・ショルトーの頭頂部に向けて射た。矢には強力な植物アルカロイドの毒素が塗られており、ストリキニーネ（strychnine）に類似した効果を持ち、筋肉の強直（tetanus）を引き起こした。これにより「サタンの笑み」（risus sardonicus）と呼ばれる顔面の痙攣が生じ、被害者は即死した。ジョナサン・スモールは屋根から侵入してロープで部屋に降り、財宝を運び出す黒幕であったが、直接手を下した殺人者ではなかった。

重要な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 遺体の顔面に固定された苦悶の笑み（risus sardonicus）と極度の筋肉収縮 第5〜6章 / 約42〜50% ワトスンはすぐさま「強力な植物アルカロイドによる死」と指摘し、ストリキニーネの効果と一致する。読者はこの時点で毒殺の手口を確認できる
2 頭皮に刺さった黒い長い棘、先端に乾いた膠質（毒素の残留物）、末端はナイフで丸く削られている 第5章 / 約42% ホームズは「毒針だ」と指摘し、ワトスンに慎重に取り出すよう求める。英国産ではない植物であり、異国の出所を示唆する
3 天井の穴、梯子、ロープ——「もう一人が屋根から侵入した」 第6章 / 約48〜50% ホームズは他のすべての侵入口を除外し、裸足の小柄な人物が上から入り、ロープで部屋に降りたと指摘する。犯行手口の全貌が明らかになる
4 義足の痕跡（窓枠の木製杭の跡）とその横にある靴跡 第6章 / 約48% 二人の共犯関係を確認：義足の男が窓の外でロープを支え、裸足の男が降りて刺殺した

2 屋根に散らばっていた毒棘の小袋（dark wooden spinesの入った小袋） 第7章 / 約58% 裸足の人物が毒吹き矢を携帯していたことを確認し、袋は慌てた逃走中に落としたものであることが分かる
3 トンガの石頭の木棒（stone-headed wooden mace） 第5章 / 約42% アンダマン島民特有の武器であり、毒吹き矢とともに同一人物を指し示している
4 屋根裏に残された完全に指が開いた裸足の足跡（各指が明確に分かれており、靴を履いた者のものではない） 第6章 / 約50% ホームズはこの細部を特に指摘し、靴を履いたことのない生活をしてきた人物であることを示唆する

偽の手がかり

# 手がかりの内容 登場箇所（章／百分率） 誤誘導の方向
1 サディアス・ショルトーの奇妙な行動：秘密の接触、警察への強い忌避、事件現場への真っ先の出現 第3〜5章 / 約25〜42% 読者は自然と彼が首謀者ではないかと疑う。アセルニー・ジョーンズもすぐに彼を逮捕し、この誤誘導に「公式の裏付け」を与える
2 バーソロミュー・ショルトーの部屋の鍵が内側からかかっていた——「密室殺人」を示唆する 第5章 / 約42% 読者はサディアスが何らかの方法で施錠したのではないかと考えるかもしれない。真実は、裸足の人物がロープで室内に降りた後、窓から侵入した共犯者が封じた

真相その二：ジョナサン・スモールとは何者か？（who is the criminal）

  • 手がかり収束点：約50%
  • 答え：ジョナサン・スモールはイギリス人で、インドの反乱期に信使アクメットを共謀して襲撃し、アグラの財宝を奪ったことによりアンダマン諸島で長年の強制労働刑に処された。彼は財宝の半分が「四つの署名」の誓約に基づく自らの正当な所有物であると信じており、モースタンとショルトーの裏切りに深い恨みを抱いていた。アンダマン諸島で島民トンガと同盟を結んだ後、脱獄してイギリスまで追跡してきた。右脚はインドでワニに噛み切られ、義足で代替している。

重要な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 モースタンの書類に記された「The sign of the four—Jonathan Small, Mahomet Singh, Abdullah Khan, Dost Akbar」という署名 第3章 / 約25% スモールの名前と「四つの署名」が初めて登場する。彼の存在を示す最初の証拠
2 ショルトー少佐の義足の白人に対する強い恐怖心、無関係の商人にさえ銃を向けるほど 第4章 / 約33% サディアスが父の生前の強迫観念を語る——義足の男がショルトーの人生における具体的な脅威であり、妄想ではないことを示唆する
3 ホームズの完全な論理的推理：モースタンの死→ショルトーの財宝獲得→スモールのアンダマン収監→ショルトーが手紙を受け取った後の死→スモールのアンダマン脱獄 第7章 / 約58% ホームズが追跡中にワトスンへ段階的に説明する。この時点で読者はスモールの正体と動機を完全に把握できる
4 アンダマン諸島の記録（最も小柄な種族、裸足、吹き矢、石棒）から裸足の人物の民族的出自を特定し、スモールが現地で服役しトンガと知り合ったことを確認 第8章 / 約67% ホームズが地名辞典を調査し、「最も小柄な種族・裸足・吹き矢」がアンダマン島民を指すと確認。スモールの経歴をさらに裏付ける

補助的な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 ロープに残された皮膚の擦り傷と血痕——「ロープ登りに不慣れな白人で、手のひらに胼胝がない」 第6章 / 約50% 義足の男がロープを無理やり登ったことを示し、職業的な船員でもなく、しかし体力は十分にある——スモールの経歴と一致する
2 モーデカイ・スミスの妻が描写する「茶色の猿のような顔の男」（トンガ）と義足の男の重い声 第8章 / 約63% スモールとトンガのコンビを確認し、二人の外見描写がより具体的になる
3 ジェイコブソン造船所でオーロラ号を発見する推理——スモールには「低い狡猾さ」（low cunning）があり、船を隠す術を知っている 第9〜10章 / 約75〜83% 読者はスモールの知能の程度を正確に評価できるようになり、後の自白時の印象と一致する

# 手がかりの内容 登場箇所（章／百分率） 誤誘導の方向
1 「小柄な謎の同伴者」の存在が長期にわたって真の民族的出自を意図的に隠されている 第6〜8章 / 約50〜67% 読者（ワトスンも含む）はまずインド人と推測し、次に南米人と推測する。ホームズが最終的にアンダマン島民であると明かすまで、ドイルは意図的にこの手がかりの収束を遅らせる

真相その三：実際に殺害を実行したのは誰か？（whodunitの核心的逆転）

  • 手がかり収束点：約85%
  • 答え：ジョナサン・スモールではなく、トンガである。スモールの目的は財宝を取り戻すことであり、本来はバーソロミューが部屋にいないことを望んでいた。しかしトンガが先に天井の開口部から毒矢でバーソロミューを射殺した。スモールは後に「あの小悪魔をロープの端で打った」と述べ、この出来事を悼んでいた。

重要な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 ホームズが第6章で既に「共犯者の犯行であり、スモールは縛ることはあっても殺すことは望まなかった」と述べているが、読者はこの細部を見落としがちである 第7章 / 約55〜58% ホームズは「ジョナサンの怒りは、バーソロミューを殺すつもりがなかったためだ」と言い、逆転への早期の示唆となっている
2 追跡中、スモールは人命よりも財宝に固執している（常に逃走を試みるが反撃はしない） 第10章 / 約83% 「彼は人を殺したくなかった」という人物像を側面から支持する
3 スモールが捕縛された後の最初の言葉：「私はショルトー氏に手を下してはいない。あの地獄の小犬トンガがやったのだ。私はそれを、肉親を失ったように悼んでいる。」 第10〜11章 / 約85〜88% 最終確認の手がかり：スモールは殺害を明確に否定し、トンガが直接の実行犯であることを確認する

補助的な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 追跡中、トンガがホームズとワトスンに向けて吹き矢を向ける（スモールではなく） 第10章 / 約83% トンガの攻撃性の高さを示し、自発的に攻撃するものであってスモールの指示によるものではないことが分かる

偽の手がかり

# 手がかりの内容 登場箇所（章／百分率） 誤誘導の方向
1 捜査の全過程を通じて「義足の男＝主犯」という枠組みが繰り返し強化される 第5〜9章 / 約42〜75% 読者は自然と首謀者こそが殺人者だと思い込む。ドイルはスモールが捕縛されるまでこの前提を覆さない
2 アセルニー・ジョーンズがサディアスと家族を逮捕して拘留し、読者は容疑者がまだ捜査中だと思い込む 第6〜9章 / 約50〜75% 警察の誤った方向が安定して読者の注意を消耗させ、ホームズがスモールの正体を宣言するまで続く

真相その四：アグラの財宝の最終的な行方（where did it go）

  • 手がかり収束点：約92%
  • 答え：スモールは追い詰められる前に、財宝を一点ずつテムズ川に沈めた。ショルトーやモースタンの子孫に恩恵が渡るくらいなら、富が川底に沈んだほうがましだと考えたのだ。「四つの署名」の誓約は彼にとって神聖なものであった——財宝は四人の共有物であり、他の誰のものでもない。

重要な手がかり

# 手がかりの内容 登場箇所（章／百分率） 説明
1 追跡中、蒸汽船オーロラ号の上でスモールが「かがんで忙しそうにし、ときおり振り向いて距離を測っている」 第10章 / 約83% 読者はこの時点では武器の準備をしていると思うかもしれないが、後から見れば財宝を川に投げ込んでいたのだ
2 鉄の箱を開けると空だった——「財宝が消えた」 第11章 / 約92% 大きな転換点：半年にわたる追跡が富の消滅という結末を迎え、同時にワトスンとメアリーの間の「財富の壁」も取り除かれる
3 スモールの自白：「それは鍵のある場所に、小さなトンガのいる場所に見つかるだろう。五マイルにわたる川底に散らばっているのだ。」 第12章 / 約96% 財宝を永久に沈めるという彼の計画が完全に明かされる

推理経路の再構築

第一の経路：犯行手口の推理（howdunit）

読者は第5章（約42%）で死亡現場と対面する：施錠された部屋、顔面をゆがめた苦悶の笑み、頭皮に刺さった細い棘。この時点で、基本的な医学知識を持つ細心の読者であれば、ワトスンとともに「植物アルカロイド毒素」という手口を推論でき、ホームズが毒針を抜き出して「英国産ではない」という特徴を検証した後、「毒吹き矢による殺人」という推論を完成させることができる。

第6章（約50%）がこの経路の完全な収束点となる：義足の痕跡と裸足の痕跡という二人組の組み合わせ、屋根からの侵入経路、ロープと金具の使用——犯行手口の全貌が一覧できる。注意深く読んでいた読者はこの時点で現場を完全に再構築できる。

第二の経路：主犯の正体の推理（whodunit — ジョナサン・スモール）

第3章（25%）の文書が最初のピースとなる：「四つの署名」、四つの人名、そのうちジョナサン・スモールだけがイギリス人である。第4章（33%）、サディアスが義足の白人に対する父の恐怖を語り、文書と一致する。第6〜7章（48〜58%）、ホームズが公開の「推理ショー」を行い、読者を論理の全連鎖へと導く。この時点ですべての重要な手がかりが揃い、注意深く追ってきた読者はホームズと同時に答えに到達できる。

第三の経路：真の殺人者の推理（whodunit — スモールではなくトンガ）

これは最も深く、最も予想外の逆転である。前半の三分の二、ホームズと読者の注意は「義足のジョナサン・スモール」に集中しており、裸足の謎の同伴者は長期にわたって「謎」として存在し、意図的にぼかされている。

最初の微かな示唆は第7章（55%）に現れる：ホームズは「あの殺害は同伴者の独立した行動で、スモールを苦しめたように思える」と述べる——しかし大半の読者は追跡の興奮の中でこの一言を読み飛ばしてしまうだろう。第10章（83%）でトンガが吹き矢を構えて銃撃されて倒れ、第11章（88%）でスモールが捕縛されて自白するまで、読者はようやく気づく：作品を通じてほぼ「付属品」として扱われてきたあの小柄な人物こそが、真の殺人者だったのだ。

これはドイルが巧妙に設計した認知のずれである：読者の視線は常に義足の男を追い、その傍らにしゃがんでいたより危険な影を見落としてしまう。

第四の経路：財宝の行方

第10章の追跡シーンで財宝がすでに消えたことを予想できる読者はほとんどいない——あのシーンの視覚的焦点はすべて追跡の緊迫感にある。空の箱の暴露は純粋な転換点であり、何の前触れもなく、しかし思いがけない感情的な解放をもたらす。

補足説明

コナン・ドイルの作品における『四つの署名』の位置づけ

『四つの署名』は第二のホームズ長編であり、『緋色の研究』（A Study in Scarlet、1887年）の後に書かれた。冒頭から意図的に「それを『緋色の研究』という小冊子にまとめた」と言及し、シリーズの世界観における自己言及を完成させている。

しかし推理の設計という点では、この作品は『緋色の研究』より格段に成熟している。『緋色の研究』の構造が「探偵の謎解き＋犯人の回顧録」という二分法であるのに対し、『四つの署名』は歴史的背景（インドの反乱、アンダマン諸島）を捜査の主線に完全に組み込み、犯人を探す過程そのものが帝国の歴史の暗部を読み解く行為となっている。

「四つの署名」——誓約の道徳的複雑性

本書の道徳的な重みは殺人そのものにあるのではなく、ジョナサン・スモールの「四つの署名の誓約」にある。これは読者が単純に非難することのできない犯人である：インドの反乱の混乱の中で不義を働き（信使の強盗に加担し）、しかしモースタンとショルトーによってより深く裏切られた。二十年の苦役に服し、マレーの財宝を前にして再び剥奪された。

ドイルはスモールに第12章で自ら語らせる機会を与える——これは稀な設計である：犯人に一人称で陳述させ、読者が探偵の総括だけを受け取るのではなく、彼の論理を体感できるようにしている。この視点の切り替えが「四つの署名」を道徳的灰色地帯の作品たらしめている。

「三段階絞り込み法」：ドイルが示す探偵推理の実演

ホームズは『四つの署名』において推理の手法を繰り返し実演しており、最も代表的なのは「消去法」の宣言である：

「不可能を排除したとき、残るもの——どれほど有り得ないことであれ——が真実に違いない。」（"When you have eliminated the impossible, whatever remains, however improbable, must be the truth."）

この言葉は第6章に登場し、ホームズの哲学の核心的な宣言である。推理学の観点からは「窮尽的排除論証」（argument by elimination）であり、探偵小説史上最も多く引用される文の一つであり、ドイルがこの作品に与えた最も永続的な遺産でもある。

帝国主義の影

『四つの署名』は植民地の歴史の扱い——というより、その扱い方そのものが分析に値する——に触れずにはいられない。トンガは作中で繰り返し「野蛮な」「獣のような」「小悪魔」と描写される。彼は作品全体を通じて言葉を発したことがなく、名前以外の人格描写を一切与えられない唯一の登場人物であり、死後はテムズ川に沈み、追悼も受けない。

これはジョナサン・スモールへの扱いと鮮明な対比をなす——スモールは丸々一章にわたって自己陳述の機会を与えられ、ホームズも彼のために弁護し、同情を示す。ドイルの帝国主義的な視野は、この不平等な対比の中に明確に現れている。

『緋色の研究』との比較

緋色の研究 四つの署名
核心的謎 沈黙の復讐（whydunit） 奪われた財宝（where + whodunit）
歴史的背景 アメリカのモルモン教 インドの反乱とアンダマン諸島
犯人像 悲劇的英雄（ジェファーソン・ホープ） 道徳的灰色の冤罪者（ジョナサン・スモール）
感情的主線 なし ワトスンとメアリー・モースタンの恋愛
推理の提示 比較的散漫 体系的（「消去法」の明確な宣言）
霧の技法 低度の誤誘導 中度：「首謀者≠殺人者」という逆転

『四つの署名』はドイルのキャリアにおける意識的な進化を示している：謎解きだけでなく、より広大な歴史的・道徳的空間の中で物語を語ることへの挑戦である。
